# Attribution

Discord YouTube Streamer is derived from [Nick McGee's Discord Music Bot](https://github.com/Nick-McGee/discord-bot).
The extracted project retains and modifies portions of that project's audio, queue, voice, YouTube, and Discord UI implementation.

The local upstream baseline used for this extraction is commit `4917444` ("Update requirements", 2023-09-23).
All staged streaming changes on top of that baseline were reviewed when this standalone repository was created.
